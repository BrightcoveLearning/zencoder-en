Error: InvalidAudioSettingsError

This topic provides details on the Zencoder API error: InvalidAudioSettingsError

Meaning

The supplied audio encoding settings were not valid. Some settings are auto-corrected unless strict mode is enabled. See the output info page for more details on which settings were invalid.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
