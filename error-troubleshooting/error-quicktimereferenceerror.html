Error: QuicktimeReferenceError

This topic provides details on the Zencoder API error: QuicktimeReferenceError

Meaning

This error means that the input file references audio or video tracks that aren't actually included in the file. So the file itself is just a placeholder that links to these outside media tracks. Since we don't have access to them – they're probably sitting on a user's hard drive somewhere – we can't process the file. To avoid this, be sure to export self-contained Quicktime files that include all media tracks.

Workarounds

Export the Quicktime file as a self-contained Quicktime files that include all media tracks.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
