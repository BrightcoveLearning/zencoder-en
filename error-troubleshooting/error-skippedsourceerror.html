Error: SkippedSourceError

Topic(s)
Troubleshooting/Error Handling
API(s)
Zencoder API

This topic provides details on the Zencoder API error: SkippedSourceError

Meaning

This error means there were no stream sources available to generate the playlist, because they were all skipped.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
