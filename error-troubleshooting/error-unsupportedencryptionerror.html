Error: UnsupportedEncryptionError

This topic provides details on the Zencoder API error: UnsupportedEncryptionError

Meaning

This error occurs because the submitted file is encrypted, preventing us from opening it. This typically means a file has DRM encryption on it, usually because the file was purchased through a source like iTunes.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
