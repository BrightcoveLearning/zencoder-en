Zencoder Error: WorkerTimeoutError

This topic provides details on the Zencoder API error: WorkerTimeoutError

Meaning

You shouldn't see this error. It means that we tried processing this file three times, and it timed out every time. Whenever we see this error, we investigate and see if there is an underlying problem that needs to be fixed. Contact us if you would like more information.

Retry this job?

Jobs with this error can be retried. Learn more about retrying jobs.
