Zencoder Error: Transfer::InvalidGranteeGroupURLError

This topic provides details on the Zencoder API error: Transfer::InvalidGranteeGroupURLError

Meaning

The S3 group account provided doesn't exist or couldn't be given permissions to the file. Check the name and make sure that it exists.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
