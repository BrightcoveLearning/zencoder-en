Zencoder Error: UnexpectedMplayerResult

Product(s)
Zencoder
Role(s)
API Developer
API(s)
Zencoder API

This topic provides details on the Zencoder API error: UnexpectedMplayerResult

Meaning

This error means there was a problem on our end. We tried processing the file, but something went wrong in the transcoding step. We'll investigate and fix the problem. Contact us if you'd like more info.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
