Zencoder Error: UploadFailureError

This topic provides details on the Zencoder API error: UploadFailureError

Meaning

This error occurs when we are unable to upload a file for a reason other than Access Denied or Server Not Found. If the server returns a response code, we will display it in the error message so that you can determine what happened and make any necessary updates to the file. Contact us to get more information about what happened.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
