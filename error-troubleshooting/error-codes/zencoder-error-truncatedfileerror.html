Zencoder Error: TruncatedFileError

This topic provides details on the Zencoder API error: TruncatedFileError

Meaning

The input file is smaller than the headers say it should be. Generally this means the file got cut off at some point previous to being transferred to Zencoder.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
