{{ page.description }}

Meaning

We tried to download a file at the specified address, but the server at that address took too long to respond. In most cases we attempt to download the file 3 times to make sure that the issue isn't a temporary connectivity issue.

Workarounds

Ensure that the file is available to users outside of your network, and that your server responds to requests for the file at the URL.

Retry this job?

Jobs with this error can be retried. Learn more about retrying jobs.
