Zencoder Error: InvalidVideoBitrateError

Product(s)
Zencoder
Role(s)
API Developer
API(s)
Zencoder API

This topic provides details on the Zencoder API error: InvalidVideoBitrateError

Meaning

The supplied video bitrate is too low for the video. Raising the bitrate should fix the job. This should only be encountered when running in strict mode, Zencoder will automatically correct for the issue otherwise.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
