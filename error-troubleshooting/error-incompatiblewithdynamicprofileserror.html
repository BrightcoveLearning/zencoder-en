Error: IncompatibleWithDynamicProfilesError

This topic provides details on the Zencoder API error: IncompatibleWithDynamicProfilesError

Meaning

This error occurs when the input media is not compatible with CAE dynamic profile generation. This could mean that the input media file has only audio and/or image data and does not contain any video tracks in it. Contact Zencoder Support if you'd like more info.

Retry this job?

Jobs with this error should not be retried. Learn more about retrying jobs.
