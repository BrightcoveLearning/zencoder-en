Encoding Settings: MP4

Topic(s)
Encoding Settings
API(s)
Zencoder API

This topic contains details for encoding settings related to MP4 outputs.

hint

hint:Boolean

API Versions: V1, V2

Parent: output

Default: false

Valid Values: true or false

Compatible Job Types: All

Example: true

Description:

Turn on hinting of MP4 files for RTP/RTSP. The default mtu_size is 1450 bytes; use the mtu_size option to change this.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "hint": true
    }
  ]
}

mtu_size

mtu_size:Integer

API Versions: V1, V2

Parent: output

Default: 1450

Valid Values: An integer between 100 and 5000

Compatible Job Types: All

Example: 2000

Description:

MTU size in bytes for MP4 hinting.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "mtu_size": 2000
    }
  ]
}
