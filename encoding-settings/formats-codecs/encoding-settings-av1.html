Encoding recommendations

AV1 is an open, royalty-free, next-generation video encoding format from the Alliance of Open Media Video. It can deliver superior quality at significantly lower bitrates compared to HEVC, VP9 and H.264 codecs.

See the AV1 Guide for more information, and pay particular attention to the Limitations for current support.

In addition to settings found on this page, you may want to include other general video settings, but note that following settings are not currently supported:

  • video_reference_frames
  • video_bframes
  • max_video_bitrate
  • quality
  • crf
  • decoder_bitrate_cap

Also note that for Resolution settings (whether you use size or width and height) both the width and height must be multiples of 8.

video_codec_level

video_codec_level:String

API Versions: V2

Parent: outputs

Valid Values:

  • 2.0
  • 2.1
  • 3.0
  • 3.1
  • 4.0
  • 4.1
  • 5.0
  • 5.1
  • 5.2
  • 5.3
  • 6.0
  • 6.1
  • 6.2
  • 6.3

Compatible Job Types: All

Example: 4.1

Description:

Constrains the bitrate and coding tree units (CTUs) (AV1).

Note that this setting can be used for other codecs with different allowed values. See Video Encoding Settings.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_level": "5.1"
    }
  ]
}

video_codec_profile

video_codec_profile:String

API Versions: V2

Parent: outputs

Valid Values: main, high, professional

Default: main

Compatible Job Types: All

Example: 4.1

Description:

Sets the encoding profile.

Note that this setting can be used for other codecs with different allowed values. See Video Encoding Settings.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_profile": "high"
    }
  ]
}
